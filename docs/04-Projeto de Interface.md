###  UC01: Criar e Gerenciar Perfil
* **RF01:** O sistema deve permitir a realização de cadastro e login diferenciado na plataforma (Doador / Instituição).
* **RF02:** O sistema deve permitir recuperação e alteração de senha.
* **RF03:** O sistema deve permitir editar informações de perfil (nome, contatos, foto, endereço).

###  UC02: Visualizar Ponto de Doação ou Instituição
* **RF04:** O sistema deve priorizar a visualização de campanhas e instituições próximas da localização do doador.
* **RF05:** O sistema deve permitir visualizar todas as instituições e campanhas por lista.
* **RF06:** O sistema deve permitir visualizar instituições e campanhas em um mapa interativo.
* **RF07:** O sistema deve permitir visualizar detalhes de um ponto (fotos, descrição, contato, tipo de doação e nota).
* **RF08:** O sistema deve permitir que doadores visualizem campanhas de uma instituição específica.

###  UC03: Filtrar Ponto de Doação ou Instituição
* **RF09:** O sistema deve permitir filtrar instituições e campanhas por localização (mapa interativo).
* **RF10:** O sistema deve permitir filtrar instituições e campanhas por tipo de doação (Alimentos, Roupas, etc.).

###  UC04 & UC05: Gestão e Compartilhamento de Campanhas
* **RF11:** O sistema deve permitir a criação e atualização de campanhas com data de início/fim, local, descrição e tipo de doação.
* **RF12:** O sistema deve permitir a visualização de quantas pessoas doaram em uma campanha.
* **RF13:** O sistema deve permitir a validação de doações realizadas.
* **RF14:** O sistema deve permitir gerar link de compartilhamento externo para campanhas/perfis.
